## Road Detection

![Demo](./demo/road.gif)

### Setting up
```
pip install -r requirements.txt
```

### Running the model
```
python detection.py -h

optional arguments:
  -h, --help            show this help message and exit
  --video_dir VIDEO_DIR
                        Path to input video
  --dim DIM [DIM ...]   Width and height of the output frame
```
